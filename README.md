DSP (Digital Systems Project): Password Cracking Tool

This is a Password Cracking Tool that is being developed for my final year project in University.

-----------------------
## Attack Algorithms 
-----------------------
- Brute Force
- Dictionary Attack
- Hybrid Dictionary Attack
- Rule-Based Attack
- RainbowTable Attack
- Markov Chain Attack

- Save and Load an attack
